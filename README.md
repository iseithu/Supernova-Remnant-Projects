This repository contains three numerical projects completed for the **Supernova Remnants** course.

## Projects

- Project 1 – Numerical Solution of the Sedov Self-Similar Blast Wave
- Project 2 – Numerical Solution of Modified Shock Equations
- Project 3 – Adapted Equipartition Calculation for SNR J0450−7050

- Jupyter Notebook
